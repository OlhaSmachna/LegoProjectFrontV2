BricksManager 2.0

Hi! My name is Olga Smachna, and this is my diploma project for IT Step Computer Academy. It presents a solution for a Lego-themed database management system featuring a backend .NET CORE API service and a frontend Angular web app. The project received some important changes since its previous desktop version: improved database structure, better query logic and cloud storage for images, resulting in a significantly improved performance.

Lego BricksManager allows you to browse the Lego parts database as a guest user or create and manage your own inventory under a signed-in account. It also allows you to export and import inventory lists in XML format supported by other major Lego building applications and online shops. And of course, you can edit database records under an account with an admin role.
